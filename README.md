## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Sadam-PK/event-management
   ```
2. Install packages
    ```sh
    npm install
    ```
    ```sh
    npm init -y server
    ```

3. Run from project directory
    ```sh
    npm run dev
    ```
